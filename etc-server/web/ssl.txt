SSL configuration on an Apache server
-------------------------------------

Many websites describe how to correctly configure a SSL web server in a secure
fashion, like http://hynek.me/articles/hardening-your-web-servers-ssl-ciphers/

Here are example configurations for ssl.example.com host with a certificate
signed by StartSSL.

Apache:
<IfModule mod_ssl.c>
<VirtualHost *:443>
    ServerName ssl.example.com
    SSLEngine on
    SSLCertificateFile /etc/ssl/ssl.example.com.crt
    SSLCertificateKeyFile /etc/ssl/ssl.example.com.key
    SSLCertificateChainFile /etc/ssl/startssl-sub.class1.server.ca.pem
    SSLProtocol ALL -SSLv2
    SSLHonorCipherOrder On
    SSLCipherSuite ECDHE-RSA-AES128-SHA256:AES128-GCM-SHA256:RC4:HIGH:!MD5:!aNULL:!EDH
    SSLCompression Off
</VirtualHost>
</IfModule>


Nginx:
server {
    listen 443;
    server_name ssl.example.com;
    ssl on;
    ssl_certificate /etc/nginx/ssl/ssl.example.com.crt;
    ssl_certificate_key /etc/nginx/ssl/ssl.example.com.key;
    ssl_session_timeout 5m;
    ssl_protocols SSLv3 TLSv1;
    ssl_ciphers ECDHE-RSA-AES128-SHA256:AES128-GCM-SHA256:RC4:HIGH:!MD5:!aNULL:!EDH;
    ssl_prefer_server_ciphers on;
}
